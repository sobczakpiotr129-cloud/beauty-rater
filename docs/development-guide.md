# 开发执行步骤

## Phase 1：项目初始化 + 文档体系 ✅ 完成
1. 创建项目目录结构
2. 安装 Python + PySide6
3. 编写标准文档（需求、技术、设计、开发指南）
4. 创建首日 dev-log
5. 编写 CLAUDE.md 指引文件
6. **验证：python main.py 启动成功**

## Phase 2：剪贴板监控 + 数据存储 ✅ 完成
1. 实现 QTimer 剪贴板轮询（500ms）
2. 检测剪贴板内容类型（文字/图片）
3. 图片压缩为缩略图（200×200px）base64 存储
4. SQLite 数据库初始化（clips 表）
5. 实现增、删、查、置顶、过期清理
6. 来源应用检测（隐私过滤用）
7. **验证：复制文字和图片，数据库有记录**

## Phase 3：主窗口 UI ✅ 完成
1. 卡片列表布局（时间降序，置顶优先）
2. 文字卡片（截断预览 + 时间戳）
3. 图片卡片（缩略图 + 时间戳）
4. 搜索栏（实时过滤）
5. 置顶/删除按钮
6. 点击卡片 → 粘贴到活动窗口
7. 淡蓝色主题
8. **验证：界面可操作**

## Phase 4：系统集成 ✅ 完成
1. 系统托盘图标 + 右键菜单
2. 全局快捷键 Alt+V（Windows 原生注册）
3. 自动粘贴（模拟 Ctrl+V）
4. 窗口失去焦点自动隐藏（Escape 快捷键）
5. **验证：托盘运行、快捷键呼出、粘贴到记事本**

## Phase 5：设置面板 ✅ 完成
1. 设置界面（齿轮图标进入）
2. 保留期限选择器（1/3/5天）
3. 隐私排除应用列表
4. 设置 JSON 持久化
5. 设置变更即时生效
6. **验证：修改设置后行为正确**

## Phase 6：测试 + 打包
1. 全功能回归测试
2. 安装 PyInstaller
3. 打包为 Windows .exe
4. **验证：干净环境安装运行**
