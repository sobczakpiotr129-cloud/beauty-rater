# 需求规格说明

## 1. 项目概述
Clipboard History — Windows 11 历史剪贴板管理工具，后台静默运行，记录用户所有复制内容，支持快速检索和回贴。

## 2. 功能需求

### 2.1 剪贴板监控
- 自动监控系统剪贴板变化，记录所有 Ctrl+C 操作
- 支持内容类型：文字（UTF-8）、图片（PNG/BMP/JPG）
- 监控间隔 ≤ 500ms，不丢数据

### 2.2 历史记录管理
| 操作 | 说明 |
|------|------|
| 查看 | 卡片式展示，按时间降序排列 |
| 搜索 | 支持文字内容搜索 + 图片文件名搜索，实时过滤 |
| 粘贴 | 点击卡片自动粘贴到当前光标位置 |
| 置顶 | 标记重要条目，置顶内容不被自动清理 |
| 删除 | 单条删除，不可恢复 |
| 保留期限 | 可设置 1天/3天/5天，过期自动清理 |

### 2.3 使用方式
- 开机自启，常驻系统托盘
- 全局快捷键 Alt+V 呼出/隐藏主窗口
- 窗口失去焦点时自动隐藏
- 托盘图标右键菜单：显示、退出

### 2.4 隐私保护
- 支持设置排除应用列表（通过进程名匹配）
- 排除的应用内复制内容不记录

### 2.5 数据存储
- 本地存储，无网络请求
- 文字直接存储，图片压缩为缩略图（≤ 200x200px）后 base64 存储
- 过期内容自动清理

## 3. 非功能需求
- 内存占用 ≤ 200MB
- 窗口呼出响应 ≤ 300ms
- 搜索过滤响应 ≤ 100ms
- 历史记录上限 2000 条
