# UI 设计规范

## 1. 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#A8D8EA` | 淡蓝色，用于标题栏、按钮、选中态 |
| 主色加深 | `#7EC8E3` | 悬浮态、卡片边框 |
| 背景 | `#F5F9FB` | 极淡蓝白，主背景 |
| 卡片背景 | `#FFFFFF` | 白色卡片 |
| 文字主色 | `#333333` | 标题、正文 |
| 文字辅色 | `#888888` | 时间戳、辅助信息 |
| 置顶标记 | `#FFB347` | 暖橙黄，置顶图标色 |
| 删除按钮 | `#E88B8B` | 柔和红色 |
| 搜索框边框 | `#C5E0EB` | 淡蓝边框 |

## 2. 窗口规格
- 窗口尺寸：380 × 520 px（宽×高）
- 圆角窗口边框
- 固定在屏幕右上角弹出（类似通知中心）
- 支持拖拽移动

## 3. 布局结构

```
┌──────────────────────────────┐
│  🔍 搜索...                  │  ← 搜索栏 (固定顶部)
├──────────────────────────────┤
│  [设置: 3天 ▼]  [清理过期]   │  ← 工具栏
├──────────────────────────────┤
│ ┌──────────────────────────┐ │
│ │ 📌 文字卡片预览...  × 🗑 │ │  ← 置顶卡片 (有📌标记)
│ │ 1分钟前                  │ │
│ └──────────────────────────┘ │
│ ┌──────────────────────────┐ │
│ │ 文字卡片预览...      × 🗑 │ │  ← 普通卡片
│ │ 5分钟前                  │ │
│ └──────────────────────────┘ │
│ ┌──────────────────────────┐ │
│ │ [缩略图]           × 🗑  │ │  ← 图片卡片
│ │ 10分钟前                 │ │
│ └──────────────────────────┘ │
│            ...               │  ← 滚动区域
└──────────────────────────────┘
```

## 4. 卡片设计
- 文字卡片：最多显示 3 行文字，超出省略号截断
- 图片卡片：左侧 48×48 缩略图，右侧显示 "[图片]" + 时间
- 卡片悬浮：轻微阴影加深 + 主色边框
- 点击卡片：直接触发粘贴操作
- 置顶卡片：左侧有 📌 图标，背景稍带暖色调
- 卡片圆角：8px，间距 6px

## 5. 交互动效
- 窗口弹出：从右上角滑入，200ms ease-out
- 卡片删除：淡出 + 上滑收起，200ms
- 搜索过滤：即时过滤，无动效延迟

## 6. 字体
- 系统默认字体栈：`-apple-system, "Microsoft YaHei", sans-serif`
- 卡片文字：14px
- 时间戳：11px，灰色
- 搜索框：14px
